# Dados

`climate_sample.csv` simula uma série climática de uma fazenda em Campo Grande/MS.

As colunas representam variáveis que poderiam vir de APIs meteorológicas, sensores de campo ou produtos derivados de observação orbital:

- `temperature_c`: temperatura media diaria.
- `humidity_pct`: umidade relativa.
- `precipitation_mm`: chuva acumulada.
- `solar_radiation_wm2`: radiação solar estimada.
- `soil_moisture_pct`: umidade do solo estimada.
- `wind_kmh`: velocidade do vento.
- `crop_stage`: fase da cultura.

O projeto usa dados simplificados para demonstrar a POC de IA. Em uma versão produtiva, a fonte seria substituída por APIs climáticas, imagens orbitais ou sensores IoT.
