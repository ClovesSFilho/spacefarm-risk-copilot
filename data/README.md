# Dados

`climate_sample.csv` simula uma serie climatica de uma fazenda em Campo Grande/MS.

As colunas representam variaveis que poderiam vir de APIs meteorologicas, sensores de campo ou produtos derivados de observacao orbital:

- `temperature_c`: temperatura media diaria.
- `humidity_pct`: umidade relativa.
- `precipitation_mm`: chuva acumulada.
- `solar_radiation_wm2`: radiacao solar estimada.
- `soil_moisture_pct`: umidade do solo estimada.
- `wind_kmh`: velocidade do vento.
- `crop_stage`: fase da cultura.

O projeto usa dados simplificados para demonstrar a POC de IA. Em uma versao produtiva, a fonte seria substituida por APIs climaticas, imagens orbitais ou sensores IoT.
